GET Endpoints

GET /api

serves up a JSON representation of all the available endpoints of the api

The Functionality this API provides is to

  • Retrieve lists of topics, articles and users
  • Access specific articles by ID, including their comment counts
  • Get comments assiociated with a particular article
  • Post new comments to articles
  • Update article vote counts
  • Delete comments

GET /api/topics

serves an array of all topics

Example Response


    {
      "topics": [{ "slug": "football", "description": "Footie!" }]
    }

GET /api/articles

serves an array of all articles

Query Parameters

Name Type Description Required Default Example
topic string filters articles by specified topic slug No - ?topic=cooking
sort_by string Sort articles by specified column
Allowed values: article_id, title, topic, author, created_at, votes, comment_count 		No created_at ?sort_by=votes
order string Determines the sort order
Allowed values: asc, desc 		No desc ?order=asc

EXAMPLE RESPONSE


    {
      "articles": [
        {
          "title": "Seafood substitutions are increasing",
          "topic": "cooking",
          "author": "weegembump",
          "created_at": "2018-05-30T15:59:13.341Z",
          "votes": 0,
          "comment_count": 6
        }
      ]
    }

GET /api/users

serves an array of all users

Example RESPONSE


    {

    }

GET /api/articles/:article_id

serves an article object based on the given article_id

Example Response


    {
      "users": [
        {
          "username": "tickle122",
          "name": "Tom Tickle",
          "avatar_url": "https://vignette.wikia.nocookie.net/mrmen/images/d/d6/Mr-Tickle-9a.png/revision/latest?cb=20180127221953"
        }
      ]
    }

GET /api/articles/:article_id/comments

serves an object containing an array of comments for the given article_id

Example Response


    {
      "comments": [
        {
          "comment_id": 89,
          "votes": 2,
          "created_at": "2020-10-24T06:08:00.000Z",
          "author": "cooljmessy",
          "body": "Esse et expedita harum non. Voluptatibus commodi voluptatem. Minima velit suscipit numquam ea. Id vitae debitis aut incidunt odio quo quam possimus ipsum.",
          "article_id": 1
        }
      ]
    }

POST Endpoint

POST /api/articles/:article_id/comments

serves a new comment to the specified article by article_id

Example Request


    {
      "username": "tickle122",
      "body": "A test comment"
    }

Example Response


    {
      "comment": {
        "comment_id": 301,
        "article_id": 1,
        "body": "A test comment",
        "votes": 0,
        "author": "tickle122",
        "created_at": "2025-06-06T17:45:27.152Z"
      }
    }
